Please note that the code will decode file that meets following conditions:
1. Needs to be in JSON format
2. Needs to have "CollectionDate" field in yyyy-mm-dd format
3. Needs to include "beachName" string
5. Needs to have a "statusFlag" field with an "UNSAFE" value
If the file does not meet the following format, feel free to update processData method in app.py to match your file.
